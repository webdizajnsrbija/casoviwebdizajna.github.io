Web Dizajn?

Web dizajner mora biti upoznat sa kodiranjem HTML i CSS, a ne programiranje kao sto mnogi misle..
Grafički dizajner, nije isto što i web dizajner.
Front-end Developer je takođe web dizajner čiji posao je da slike pretvori u pokret.
Što se tiče Back-end Web Developera, sama reč “back” kaže čime se on bavi . Uglavnom serverom, izradom aplikacija, bazom podataka i sl. Osoba koja je “zadužena” za ono što ne vidimo ali je itekako bitna da bi sve radilo bez premca.
Koristi: PHP, Ruby, Python i sl.

Html?

HTML (HyperText Markup Language) je veoma jednostavan jezik koji služi za izvršavanje programa na daljinu.
HTML je skelet koji drži ceo sajt.
HTML nije programski jezik.

Za kreiranje HTML dokumenta je dovoljan bilo koji tekst editor kao na primer (Notepad++, Dreamweaver, NetBeans, Coda, itd.)

Css?

CSS (Cascading Style Sheets) je jezik formatiranja pomoću koga definišemo izgled internet strana.
Web dokumenti se dele na:
HTML deo koji definiše sadržaj internet stranice (skelet),
CSS deo koji definiše izgled i strukturu internet stranice (kao na primer boja slova, pozadine, margine i tako dalje).

Seo-optimizacija?

Prvi Na Googlu
Veoma veliki procenat ljudi koji nešto traže na internetu pretragu vrši pomoću pretraživača. Izbacivanjem rezultata na pretraživaču skoro 90% ljudi pregleda samo prvu ili drugu stranu rezultata pretrage. To je razlog zašto postoji SEO.
Cilj je da što boljom optimizacijom postignete visok rang na pretraživačima, a samim tim dobijate mnogo novih posetilaca.
Kompanija Google je propisala preko 200 pravila i smernica koje svaki vebsajt mora da ispuni kako bi se dobro rangirao u pretrazi.

Media-Queries?

Web sajtovi koji su optimizovani samo za pregled preko desktop ili laptop računara, znaju da budu jako neugodni za korišćenje od strane posetioca koji sajtu pristupaju preko uređaja sa manjim ekranima, jer je potrebno skrolovati ili zumirati stranicu da bi se jasno videle određene stvari.
Media-Queries nam omogućava da jednu web stranicu možemo pregledati na raznim uređajima različitih rezolucija, na taj način što elementi koji se nalaze na web stranici menjaju svoju poziciju, veličinu i izgled u zavisnosti od veličine ekrana uređaja, odnosno prilagođavaju se ekranu korisnika.
S obzirom da u današnje vreme sve više ljudi koristi mobilne telefone i tablete za pregled web sajtova, prilagodljiv web dizajn se nameće kao obavezna opcija kod izrade web sajtova.
